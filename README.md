# Cross-Domain-Transfer-Learning-Based-Intelligent-Diagnosis-for-High-Speed-Train-Axle-Box-Bearings
# 数智赋能——跨域迁移学习高速列车轴承智能故障诊断
## 一、项目概述
本项目完整实现振动信号预处理、多域特征提取、多模型对比、跨域迁移诊断全流程，解决实验室台架数据与真实高铁轴箱轴承数据分布不一致、实车故障标注稀缺行业难题。融合轴承故障物理机理与机器学习、迁移学习算法，可区分正常、内圈故障、外圈故障、滚动体故障四类轴承状态，全部代码基于MATLAB编写。

本项目面向高速列车轴箱轴承实测无标签、跨工况诊断痛点，搭建机理与数据融合的跨域迁移智能诊断体系。以公开台架振动信号为源域，列车实测数据为目标域，采用巴特沃斯滤波、Z-score 标准化降噪，融合时域、频域、BPFO/BPFI 故障机理特征，经 PCA 降维得到 31 维核心特征。对比 CNN、LSTM、SVM 等算法，选定随机森林作为源域基准模型，诊断准确率 83.3%。针对域分布偏移，提出 MMD 特征对齐、模型微调、半监督伪标签复合迁移方案，域差异大幅缩减。实测结果表明，相比无迁移方法整体精度提升 25%，目标域三类故障识别率提升 20%~30%，领域适应性得分达 0.9，可为轨道交通小样本跨域故障监测提供完整可落地技术方案。


## 二、数据集说明
1. 源域数据集：标准轴承台架振动数据，共124条样本，覆盖0~3HP载荷、多故障尺寸，采样频率12kHz/48kHz，标签完整；
2. 目标域数据集：真实高速列车轴箱实测数据，16条无标签样本，车速90km/h，采样32kHz，用于迁移验证。

## 三、核心功能模块
1. 信号预处理：巴特沃斯带通滤波、去趋势、Z-score标准化、滑窗样本扩充；
2. 多维特征工程：时域/频域/时频域/故障机理特征提取，PCA降维去冗余；
3. 源域基准建模：随机森林、SVM、KNN、1D-CNN、LSTM训练与指标对比；
4. 跨域迁移算法：MMD特征对齐、协方差CORAL对齐、模型微调、半监督伪标签训练；
5. 可视化分析：特征箱线图、相关性热图、PCA聚类、混淆矩阵、域分布对比图；
6. 量化评估：准确率、精确率、召回率、F1、MMD域差异、领域适应性得分。

## 六、核心实验结果
1. 源域最优模型随机森林：准确率83.3%，宏F1=0.85；
2. 复合迁移方案域适应性得分0.9，诊断精度较直接预测提升25%；
3. 无标签实车数据故障识别率提升20%~30%，类别预测均衡无严重偏置。

## 七、项目优缺点
### 优势
1. 机理+数据双驱动，特征物理可解释，避免纯黑盒模型；
2. 多层域对齐+半监督策略，适配工业无标注实测场景；
3. 完整可视化输出，便于结果展示与论文绘图。
### 局限
1. 集成迁移模块计算量偏大，暂不满足毫秒级在线实时监测；
2. 半监督效果依赖高置信伪标签，极低信噪比下性能小幅下降。

## 八、运行步骤
1. 将源域、目标数据集放入代码指定路径；
2. 依次运行Task1、Task2、Task3脚本；
3. 所有图表、指标csv、文本报告自动输出至result文件夹。
